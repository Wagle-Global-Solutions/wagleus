 Integration with Legacy Systems

                    
Useful when working with:

                    
    
                        
  • SOAP web services

  • Enterprise systems requiring XML
    • 
                        
  • EDI (Electronic Data Interchange) processes
    • 
                    

                    

                        
// Modern API Response
{
  "order": {
    "id": "12345",
    "items": ["item1", "item2"]
  }
}

â†’ Legacy System Compatible XML:

  <order>

    <items>

      <item>item2</item>
    </items>
  </order>
</root>
